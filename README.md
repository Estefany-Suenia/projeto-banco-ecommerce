# Projeto Lógico de Banco de Dados – E-commerce

Este é um projeto criado como desafio do bootcamp da DIO, com o objetivo de praticar modelagem lógica e consultas SQL em um banco de dados para um sistema de e-commerce.

## Objetivo

Criar um modelo lógico, com tabelas para clientes (PF e PJ), fornecedores, produtos, pedidos, itens dos pedidos, pagamentos e entregas.

## Estrutura do projeto

- `script_ddl.sql`: Criação das tabelas no banco de dados.
- `script_dml.sql`: Inserção de dados para testes.
- `queries.sql`: Consultas usando SQL.
- `modelo.png`: imagem do modelo lógico do banco (relacionamentos entre as tabelas).

## Consultas realizadas

- Quantos pedidos foram feitos por cada cliente?
- Produtos com seus fornecedores e estoques
- Fornecedores que também são clientes
- Valor total de cada item do pedido
